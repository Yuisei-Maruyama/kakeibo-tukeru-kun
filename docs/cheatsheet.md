# 家計簿Bot コマンド一覧

## 🔰 基本の使い方

**画像を送るだけで自動登録！**
- レシート写真を送信 → 自動で金額・店舗を解析して登録

---

## 📊 情報を見る

### 残高を確認する
```
@残高
```

**返答例:**
```
💰 外食残高

田中: ¥35,000
鈴木: ¥42,000

📅 今月の予算: ¥50,000/人
```

### 履歴を見る
```
@履歴
```

**返答例:**
```
📋 直近の支出履歴

1. 12/03 [外食費用] 田中 ¥1,280
2. 12/02 [買い物費用] 鈴木 ¥2,500
3. 12/01 [外食費用] 田中 ¥980
```

### 今月の集計を見る
```
@集計
```

**返答例:**
```
📊 家計簿レポート（12/1〜12/31）
👤 外食担当: 田中

【🍽️ 外食費用】
田中: ¥20,000
鈴木: ¥3,000
合計: ¥23,000

【🛒 買い物費用】
田中: ¥15,000

合計: ¥27,000

【🧳 旅行費用】
田中: ¥35,000
鈴木: ¥28,000
合計: ¥63,000
```

### 過去の月の集計を見る
```

```

## 💰 支出を登録する

### 基本（自分の支出）
```
@追加 @自分 外食費用 1280
```

**返答例:**
```
✅ 登録しました！
📝 外食費用: ¥1,280（鈴木）
🏪 手動入力
📅 2026/01/04

💰 鈴木の外食残高: ¥48,720
```

### 日付を指定する
```
@追加 @自分 外食費用 1280 12/15
```

### 他の人の分を登録する
```
@追加 田中 外食費用 1280
```

### 買い物を登録する

```
🛒 買い物費用として登録されました
```

## 💡 便利な使い方

### `@自分` を使う
自分の名前を自動入力してくれます
```
@追加 @自分 外食費用 1280
```

### 日付を省略する
日付を書かないと「今日」になります
```
@追加 @自分 外食費用 1280
```
↑これで今日の日付で登録されます

### 対話形式で入力する
コマンドだけ送ると、質問に答えていく形式で入力できます
```
@追加
```
```
@削除
```
```
@予定
```

### 間違えたらキャンセル
対話形式の途中でキャンセルできます
```
@キャンセル
```

---

## 📅 日付の書き方

| 書き方 | 意味 | 例 |
|--------|------|-----|
| 書かない | 今日 | `@追加 @自分 外食費用 1280` |
| `12/15` | 今年の12月15日 | `@追加 @自分 外食費用 1280 12/15` |
| `2024/12/15` | 2024年12月15日 | `@追加 @自分 外食費用 1280 2024/12/15` |

## ❓ トラブルシューティング

### 画像が認識されない
→ もう一度、鮮明な写真を送ってみてください

### 残高がマイナスになった
→ 予算を超えています。`@残高` で確認してください

### 間違えて登録した
→ `@削除` コマンドで削除できます

### コマンドがうまく動かない
→ `@ヘルプ` で詳しい使い方を確認してください

## 🚀 クイックスタート

**初めて使う方は、この3つだけ覚えればOK！**

1. **画像を送る** → 自動で登録される
2. **`@残高`** → 残高を確認
3. **`@集計`** → 今月の使用額を確認

**慣れてきたら:**
- **`@追加 @自分 外食費用 金額`** → 手動で追加
- **`@削除 @自分 外食費用 金額`** → 削除
